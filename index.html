This v3 script writes normalized event + payment currency fields to one "Events" tab and de-duplicates retries using event IDs.
Optional: send "Email artist for payment" from the app, not your Gmail
By default this email sends from the Google account that owns the script (your Gmail). To make it come from a neutral address instead, sign up for a transactional email provider — Resend, Brevo, SendGrid, Mailgun, or Postmark — and verify a sender address. Then in the Apps Script editor open Project Settings → Script properties and add:
  • MAIL_PROVIDER — one of resend, brevo, sendgrid, mailgun, postmark
  • MAIL_API_KEY — your provider's API key (stays on Google's servers, never in this app)
  • MAIL_FROM — the verified sender, e.g. noreply@yourdomain
  • MAIL_FROM_NAME — display name (optional, e.g. Lyrical Inventory)
  • MAIL_REPLY_TO — where replies should go (optional; leave blank to keep your address hidden)
Mailgun also needs MAIL_DOMAIN (and MAIL_REGION=eu for the EU region). Redeploy after saving. If these aren't set, the email still sends from your Gmail as before.
